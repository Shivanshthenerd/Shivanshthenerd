# 📈 Indian SIP Churn Rate Prediction

A **production-grade** Python project for predicting SIP (Systematic Investment Plan) churn in India, built on real AMFI mutual fund data and a realistic simulated investor dataset.

---

## Quick Start

```bash
pip install -r requirements.txt
python run_pipeline.py
```

The pipeline runs all 6 stages and produces `data/features/features.csv` — the primary ML-ready dataset.

---

## Pipeline Overview

| Stage | Module | Description |
|---|---|---|
| **0** | `simulate_investors.py` | Generate NIFTY 50, fund NAV, and investor data (idempotent) |
| **1** | `data_loader.py` | Load AMFI funds + monthly panel |
| **2a** | `data_cleaning.py` | Impute, cap, dedup, merge AMFI data |
| **3** | `feature_engineering.py` | AMFI feature set → `features_amfi.csv` |
| **4** | `data_loader.py` | Load NIFTY, fund NAV, investor data |
| **5** | `investor_cleaning.py` | Expand timelines, distribute missed payments, SIP math, join NAV + NIFTY |
| **6** | `feature_engineering.py` | Investor feature set → `features.csv` |

## Investor Data Schema (`sip_investors.csv`)

| Column | Description |
|---|---|
| `investor_id` | INV00001 … INV05000 |
| `investor_name` | Realistic Indian name (e.g. Rahul Sharma) |
| `city` | Mumbai, Delhi, Bangalore, … (18 cities) |
| `age_group` | 25-35 / 36-45 / 46-55 / 55+ |
| `fund_name` | One of 20 real Indian MF names |
| `monthly_investment` | ₹500 – ₹50 000 (log-normal, rounded to ₹500) |
| `sip_start_date` | Jan 2015 – Dec 2022 (random) |
| `sip_end_date` | Stop date for churned investors (blank if active) |
| `missed_payments` | Total missed SIP instalments |
| `status` | active / stopped |
| `churn` | **Target**: 1 = stopped, 0 = active |

## Feature Engineering (`data/features/features.csv`)

**268 371 rows × 44 columns** — investor × month panel, 0 nulls, churn = 43 %

| Group | Key features |
|---|---|
| **Investor behaviour** | `missed_payment_ratio_12m`, `payment_streak`, `consec_missed`, `avg_monthly_investment`, `tenure_band`, `is_early_stage` |
| **CAGR** | `cagr`, `cagr_vs_nifty`, `cagr_positive`, `cagr_band` |
| **Max drawdown** | `portfolio_drawdown`, `max_drawdown`, `in_drawdown`, `drawdown_recovery` |
| **Risk-adjusted return** | `sharpe_12m`, `sortino_12m`, `calmar_ratio` |
| **NIFTY context** | `market_regime`, `fund_vs_nifty_3m`, `fund_vs_nifty_12m`, `nifty_drawdown_flag` |
| **Fund characteristics** | `fund_category`, `expense_ratio`, `fund_monthly_return` |
| **Portfolio** | `portfolio_value`, `total_invested`, `cumulative_units`, `cumulative_missed` |
| **Target** | **`churn`** (1 = discontinued SIP, 0 = active) |
